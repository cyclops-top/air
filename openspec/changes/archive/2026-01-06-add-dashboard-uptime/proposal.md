# Proposal: 在 TUI 仪表盘增加运行时长显示

## Why
目前 TUI 仪表盘展示了基本的系统状态和下载统计，但用户无法直观地看到服务已经连续运行了多久。增加一个实时更新的运行时长（Uptime）计时器可以提供更好的监控反馈。

## What Changes
- **状态管理**: 在 `Stats` 结构体中记录服务启动的时间戳。
- **UI 渲染**: 在 `src/dashboard.rs` 的 Header 区域增加 "Uptime" 字段，显示从启动至今的时长。
- **统计摘要**: 在程序退出后的最终统计打印中，也增加总运行时长的显示。

## Impact
- **监控增强**: 提供更完整的服务状态监控。
- **统计完整性**: 最终摘要包含了时间维度的统计。
