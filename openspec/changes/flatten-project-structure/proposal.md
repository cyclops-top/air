# Proposal: 展平项目结构

## 摘要
目前 Rust 项目代码位于二级目录 `air/` 中。本提案建议将 `air/` 目录下的所有内容移动到项目根目录，以简化目录结构，使根目录直接作为 Rust 项目的根路径。

## 目标
1.  **简化结构**：移除不必要的 `air/` 中间目录。
2.  **根目录即项目**：使 `Cargo.toml`、`Makefile` 和 `src/` 直接位于仓库根目录。
3.  **保持功能**：确保移动后构建、测试和运行命令依然有效。

## 范围
*   **文件移动**：将 `air/` 下的所有文件和目录（包括隐藏文件如 `.gitignore`, `.git`）移动到根目录。
*   **清理**：移除移动后的空 `air/` 目录。
*   **配置更新**：如有必要，更新相关配置中的路径。
