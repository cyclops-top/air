# UI Requirements

## ADDED Requirements

### Requirement: 全屏 TUI 仪表盘
程序启动后 MUST 进入全屏 TUI 交互模式，以实时监控服务器状态。

#### Scenario: 仪表盘布局
当进入 TUI 模式时：
- 屏幕顶部 MUST 固定显示：共享目录路径、局域网及本地访问地址、当前统计数据（文件数、总流量）。
- 屏幕中部/下部 MUST 提供可滚动区域显示运行时日志。
- 屏幕底部 MUST 显示操作指引（如 'q' 键退出）。

#### Scenario: 日志滚动
用户可以使用键盘（如方向键）在日志区域内上下滚动查看历史记录。

### Requirement: 退出后统计展示
在 TUI 模式结束后，程序 MUST 将运行统计数据打印到常规标准输出。

#### Scenario: 打印摘要
程序关闭后，控制台最后输出的内容 MUST 包含：
- 总计下载文件数。
- 总计下载字节量（格式化后的）。
